React Hooks

A Little Background

  • Before we launch into hooks, I'll give a little background on why we might we might want hooks at all.
  • The main impetus for hooks is class-based components vs functional components.

Functional Components

  • Use ES2015 classes that inherit from React to create components.
  • They allow a component to manage its own state, as well as define how it behaves at different points in its life.

Class-based Components

  • Use ES2015 classes that inherit from React to create components.
  • They allow a component to manage its own state, as well as define how it behaves at different points in its life.
